### Live demo: https://responsiveaudio.vercel.app/

## Features:
- You upload a audio file and click on "play" button.
- The mesh sphere in the center reacts on the music as it plays.
- On the top right corner you have a control panel by which you can control the Bass, Mid, Trebel, Volume and Sensitivity.
- Also the background particles too react on the music beats.

## Tech used:
- THREE JS
- Javascript
- WebGLRenderer
- THREE JS's apis
